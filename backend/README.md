# @mimopo/bgpg-backend

This is a [Nest](https://github.com/nestjs/nest) application using [Express](https://expressjs.com) & [Socket.io](https://socket.io).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# production mode
$ npm run start-prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test-e2e

# test coverage
$ npm run test-cov
```
